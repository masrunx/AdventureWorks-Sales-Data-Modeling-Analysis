# AdventureWorks-Sales-Data-Modeling-Analysis
هدف هذا المشروع إلى تحويل بيانات مبيعات ضخمة وغير منظمة إلى نظام تحليل متكامل لدعم اتخاذ القرار. تم العمل على دورة حياة البيانات كاملة، بدءاً من تنظيف البيانات الخام وصولاً إلى بناء نموذج بيانات (Data Model) احترافي واستخراج رؤى تحليلية تجيب على أسئلة العمل الاستراتيجية.
